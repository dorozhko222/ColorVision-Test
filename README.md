# 👁️ ColorVision Test

Тест на дальтонизм с таблицами Рабкина + запись на бесплатную консультацию к офтальмологу.

## 🚀 Запуск
1. Установи зависимости: `pip install -r requirements.txt`
2. Запусти: `python app.py`
3. Открой: http://127.0.0.1:5000

## 📁 Структура
- `app.py` — сервер на Flask
- `test_game.py` — тест на Pygame
- `static/images/` — таблицы Рабкина

## 📊 Админка
http://127.0.0.1:5000/admin
